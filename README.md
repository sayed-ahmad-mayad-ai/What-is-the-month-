# Month Number Converter

A simple Python program that converts month numbers (1-12) to their corresponding month names.

## Description

This program prompts the user to enter a month number between 1 and 12, and displays the corresponding month name. It includes input validation to ensure the user enters a valid month number.

## Usage

Run the script using Python:

```bash
python test.py
```

When prompted, enter a number between 1 and 12 to see the corresponding month name.

## Example

```
Enter month number (1-12): 3
The month is: March
```

## Requirements

- Python 3.x

## Files

- `test.py` - Main program file containing the month conversion logic
